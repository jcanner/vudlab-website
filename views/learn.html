Getting started with d3.js

I'm assuming you know nothing about programming. Don't get worried if something is confusing, and DON'T bang your head against a wall trying to figure things out. Check if someone on stack overflow has already had the same problem, ask someone at VUDlab, or just move on and keep reading.

Step Zero: Get excited

Browse through this massive compendium of d3.js examples, and check out the work of the master himself, Mike Bostock, who created d3.js and now makes all those sweet data visualizations at the New York Times. YOU can be the person who takes data visualization into your research field, your company, or your nonprofit. What's more, learning data visualization is also just a great introduction to programming...a way to get motivated while mastering 'while loops.'

Step One: Setup

Make sure you're either using a Mac or running Linux on your PC. I know...I thought programmers were all snobs, too, when they told me I couldn't use Windows. But unfortunately it really is extremely difficult to code in Windows. So if you have a PC, install Ubuntu, which is a type of Linux operating system. Next, install Sublime Text, which is a text editor that makes programming way easier. Finally, install Sublime package control, which is a way to get awesome add-ons to Sublime that make it even easier to edit code.

Step Two: Get your feet wet

The single best way to get started programming in d3.js is with the Dashing D3 tutorial (~2 hrs). You don't really have to know programming to do it.

Step Three: Back to basics

Now you need to learn some little JavaScript. Do lessons 1-6 in Code Academy's JavaScript course. The 'Web Design Fundamentals' course is also a great introduction to html and css, but it's not necessary to know much web design to use basic d3.js.

Step Four: Immortal technique

Go through the d3.js workout series we've prepared. They're based on selections from a great video tutorial series made by a programmer with the handle enjalot. He's using tributary. Once you've watched, see if you can recreate what enjalot is doing without looking like you did in the workouts.

Step Five: Practice practice

If you need to review, then check out Scott Murray's tutorial series. If not, go ahead and pick some basic charts from Mike Bostock's list of basic d3 charts. Look how it's structured and see if you can recreate them.
